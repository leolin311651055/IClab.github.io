這次 APR 課程是我第一次接觸實際的 physical design 流程，挑戰蠻多的。以前在 RTL 階段，主要是關心功能正確，但在 APR 裡，wire 的電阻電容、timing closure、power delivery、signal integrity 這些現象都必須面對。從 floorplan、power planning、placement、CTS 到 routing，每一步都讓我體會到物理實現和設計邏輯之間的落差，也感受到 layout 規劃對效能與可靠度的重要性。雖然剛開始覺得複雜，但一步步操作下來，會發現每個動作背後都有它的邏輯與物理意義。

整體來說，APR 課程不只是學工具操作，更是訓練我把抽象設計轉化為實際可行 layout 的能力。這讓我第一次真實感受到從 RTL 到晶片實現的完整流程，也理解到未來做設計時，必須同時考慮 timing、power、routing 和 system integration，這些才是實際落地時最棘手但也最有意思的部分。
